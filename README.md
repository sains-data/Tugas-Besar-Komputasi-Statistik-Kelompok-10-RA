## ğŸ“ Deskripsi Penelitian

Penelitian ini bertujuan menganalisis pola dan ketimpangan **putus sekolah siswa SD** berdasarkan:

* Kecamatan
* Gender (Laki-laki vs Perempuan)
* Status sekolah (Negeri vs Swasta)

Data bersumber dari **Portal Data Pendidikan Kemendikbud Ristek** dan divisualisasikan melalui:

* Barplot
* Grouped Barplot
* Stacked Barplot
* Barplot komposisi dan agregat

Visualisasi membantu mengungkap pola distribusi siswa putus sekolah, seperti:

* Kecamatan dengan angka tertinggi adalah **Natar**
* Laki-laki cenderung lebih rentan putus sekolah
* Sekolah negeri menyumbang kasus lebih banyak dibanding swasta

Hasil analisis diharapkan menjadi dasar rekomendasi kebijakan pendidikan daerah.
